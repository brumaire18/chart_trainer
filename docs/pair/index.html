  • max_drawdown: 資産曲線の最大ドローダウンです。深いほど減点します。
  • sum_total_pnl: 最適化後ペア群の合計損益です。
  • best_total_pnl: 最良1ペアの損益です。
  • avg_win_rate: 最適化後ペア群の平均勝率です。
  • watchlist_count / candidate_count: 探索幅と監視候補の厚みです。
